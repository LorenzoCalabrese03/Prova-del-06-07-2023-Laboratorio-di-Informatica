◆ acquisisciNomeFile()

char * acquisisciNomeFile ( char *  nomeFile)

Acquisisce il nome del file di input dall'utente.

Parametri
[in]*nomeFileIl puntatore al buffer in cui memorizzare il nome del file. Deve essere inizializzato con una dimensione sufficiente per contenere il nome del file.
Precondizione
Il nome del file viene controllato dalla funzione validatore_file.
Postcondizione
Il nome del file di input viene acquisito correttamente dall'utente e restituito come stringa.

◆ gestoreScelta()

void gestoreScelta ( Pila *  pila)

Gestisce la scelta dell'utente e esegue le operazioni corrispondenti.

Parametri
[in]*pilaLa pila di dati su cui eseguire le operazioni.
Precondizione
La pila deve essere valida e correttamente inizializzata.
Postcondizione
Le operazioni selezionate dall'utente vengono eseguite correttamente sulla pila. La memoria allocata per le pile dopo l'utilizzo viene rilasciata.
